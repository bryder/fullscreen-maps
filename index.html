Add the missing fullscreen button

On some sites that features maps there is no possibility to expand the map from a small box to fill the entire screen. These bookmarklets and user scripts will add the missing button to make that possible.

There are two alternative ways to use these, as bookmarklets that can be selected each time you visit a maps page and want to add a fullscreen button or as user scripts that will run automatically every time you visit a maps page.

Available for these sites:

  • Garmin Connect
  • Eniro

Add a feature request as an issue on GitHub.
